instructions
============

Make sure you have [Node.js] (http://www.nodejs.org) and
[TypeScript] (http://www.typescriptlang.org/) installed.

Compile your TypeScript code with
`tsc --out OUTPUT_FILE INPUT_FILE`
or 
`tsc --out OUTPUT_FILE --watch INPUT_FILE`
